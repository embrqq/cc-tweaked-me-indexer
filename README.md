# CC: Tweaked and Applied Energistics ME Indexer

A web server and programs that can be run against an ME system in modded minecraft. Computes and displays rates of production/consumption, provides an interface for scheduling crafts, and generally a way to get insights into your minecraft resources while outside of the game client.

## Setup

### CC: Tweaked Settings

In order for the turtles to communicate with the web-API, the HTTP setting must be enabled in CC: Tweaked settings.
